## 🛠️ Local Development Setup

### Steps to Set Up the Project Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/TobiasPfeil02/hangman-game.git
   cd hangman-game 
   ``` 
2. Install Dependencies:
    ```bash
    npm i
    ``` 
3. Start the development server:
    ```bash
    npm run dev 
   ```
